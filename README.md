# 🐻‍❄️ Project: Students Manager System 🐻‍❄️

## ► Description

A small system wil be developed to allow the registration of students, with their respective information. Besides that, the system also allows checking the whole **students list**, **search for an specific student** by his id, **update specific student information**, or **delete its information at all**.

---

# ► Systems's features

## - Feature 1 — Register a new Student [DONE ✅]
Allow registering students in the system.

## - Feature 2 — Check Students List [DONE ✅]
Allow checking the whole students list if there is one available.

## - Feature 3 — Check specific Student Information [DONE ✅]
Allow checking specific's student information.

## - Feature 4 — Update Student Information [DONE ✅]
Allow updating student information anytime.

## - Feature 5 — Delete Student Information [DONE ✅]
Allow deleting students from the students list.
